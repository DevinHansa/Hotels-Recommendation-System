# Hotel Recommendation System

Welcome to the Hotel Recommendation System! This project provides personalized hotel recommendations based on user location and trip description, leveraging natural language processing (NLP) techniques.

## Features

- **Personalized Recommendations**: Suggests hotels based on the user's trip description and location.
- **NLP Techniques**: Utilizes tokenization, stop word removal, and lemmatization.
- **Similarity Scoring**: Ranks recommendations by similarity to user input and average score.
- **Data-Driven**: Uses real hotel review data for accurate recommendations.

## Dataset

This project uses the [515K Hotel Reviews Data in Europe](https://www.kaggle.com/jiashenliu/515k-hotel-reviews-data-in-europe) dataset from Kaggle. The dataset contains reviews, scores, and other details for hotels across Europe.

## Requirements

To run the hotel recommendation system, install the following libraries:

- `pandas`
- `numpy`
- `nltk`
